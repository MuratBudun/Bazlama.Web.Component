# Bazlama Web Component - Core Examples

Basic usage examples demonstrating the core features of Bazlama Web Component.

## Live Demo

**[View Live Demo](https://muratbudun.github.io/Bazlama.Web.Component/sample-core/)**

## Features Demonstrated

- Basic Web Component creation
- Property and Attribute management
- Event handling
- DOM updates with hooks

## Local Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
