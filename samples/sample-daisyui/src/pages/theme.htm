Baz-Theme-Switcher Component

Dropdown component for switching between DaisyUI themes

Features

  • Support for 11+ DaisyUI themes
  • Save theme preference with LocalStorage
  • Automatic theme application
  • Optional label display
  • Customizable theme list

Usage

Basic Usage

<baz-theme-switcher></baz-theme-switcher>

Without Label

<baz-theme-switcher show-label="false"></baz-theme-switcher>

Custom Theme List

<script>
  const switcher = document.querySelector('baz-theme-switcher');
  switcher.Themes = ['light', 'dark', 'cupcake', 'cyberpunk'];
</script>

Properties

Property Type Default Description
CurrentTheme string "nord" Currently active theme
Themes string[] ["nord", "abyss", ...] List of available themes
ShowLabel boolean true Label visibility

TypeScript Example

import BazThemeSwitcher from "./baz-ui/baz-theme/baz-theme-switcher"

// Select component
const switcher = document.querySelector('baz-theme-switcher') as BazThemeSwitcher

// Change theme programmatically
switcher.CurrentTheme = "cyberpunk"

// Set custom theme list
switcher.Themes = ["light", "dark", "cupcake"]

// Hide label
switcher.ShowLabel = false

LocalStorage

The component automatically saves the user's selected theme in localStorage with the "theme" key. When the page is reloaded, the last selected theme is automatically loaded.

This component is fully developed using the Bazlama Web Component framework!
