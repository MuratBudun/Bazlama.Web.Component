Tab Component Overview

Baz-Tab is a flexible and powerful tab component that supports dynamic tab management.

  • Add/Remove tabs dynamically
  • Icon support
  • Closable tabs
  • Reactive updates

Key Features

Dynamic Tabs
Icon Support

Reactive

Usage Example

<baz-tab closable>
  <div tab-id="tab1" title="Tab 1" icon-name="home">
    Content 1
  </div>
  <div tab-id="tab2" title="Tab 2">
    Content 2
  </div>
</baz-tab>

Properties & Attributes

Property Attribute Type Default Description
ActiveTabId active-tab-id string "" Currently active tab ID
Closable closable boolean false Enable close buttons on tabs

Methods

addTab()

Adds a new tab to the component

addTab(tabId: string, title: string, content: string, iconName?: string, activate?: boolean): void
Parameters:
  • tabId - Unique identifier for the tab
  • title - Tab title
  • content - HTML content
  • iconName - Optional icon name
  • activate - Activate after adding (default: true)

removeTab()

Removes a tab from the component

removeTab(tabId: string): void

updateTab()

Updates an existing tab

updateTab(tabId: string, title?: string, content?: string, iconName?: string): void

clearTabs()

Removes all tabs

clearTabs(): void

getTabCount()

Returns the number of tabs

getTabCount(): number

hasTab()

Checks if a tab exists

hasTab(tabId: string): boolean

Dynamic Tab Management

const tabComponent = document.querySelector('baz-tab');

// Add a new tab
tabComponent.addTab('settings', 'Settings', '<div>Settings content</div>', 'settings');

// Update existing tab
tabComponent.updateTab('settings', 'App Settings');

// Remove a tab
tabComponent.removeTab('settings');

Advanced Features

<baz-tab id="myTabs" closable active-tab-id="profile">
  <div tab-id="profile" title="Profile" icon-name="user">
    <div class="p-4">
      <h3>User Profile</h3>
      <!-- Profile content -->
    </div>
  </div>
  <div tab-id="settings" title="Settings" icon-name="settings">
    <div class="p-4">
      <h3>Settings</h3>
      <!-- Settings content -->
    </div>
  </div>
</baz-tab>

<script>
const tabs = document.querySelector('#myTabs');

// Check tab existence
if (tabs.hasTab('profile')) {
  console.log('Total tabs:', tabs.getTabCount());
}

// Listen to active tab changes
tabs.addEventListener('property-changed', (e) => {
  if (e.detail.propertyName === 'ActiveTabId') {
    console.log('Active tab changed to:', e.detail.value);
  }
});
</script>
