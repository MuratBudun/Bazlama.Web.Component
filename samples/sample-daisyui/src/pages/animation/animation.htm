BazAnimation Utility

Web Animations API wrapper with 18+ built-in animations

Quick Controls

Overview

BazAnimation is a powerful animation utility built on top of the Web Animations API. It provides a clean, Promise-based interface for creating smooth animations with built-in accessibility support and performance optimization.

Key Features

  • 18 built-in animations (fade, slide, zoom, bounce, shake, rotate, pulse, flash, flip)
  • Promise-based async API for animation sequencing
  • Parallel animation support
  • Custom animation registration
  • Reduced motion accessibility support
  • Global playback speed control
  • Callback support for animation completion

Configuration API

The controls above demonstrate the configuration API. These settings affect all animations globally. 

BazAnimation.configure({
  enabled: true,        // Enable/disable all animations
  playbackSpeed: 1.0,   // Speed multiplier (0.1 - 3.0)
  reduceMotion: true    // Respect system preferences
});

Accessibility Support

BazAnimation respects the user's prefers-reduced-motion setting. When enabled, animations will be skipped automatically to prevent motion sickness and improve accessibility for users with vestibular disorders.

Try enabling "Reduce Motion" in your operating system's accessibility settings and see how animations are automatically disabled.

Sequence Animations

Run multiple animations one after another

Watch Me! 
await BazAnimation.sequence(element, [
  { name: 'fadeIn', options: { duration: 300 } },
  { name: 'bounce', options: { duration: 500 } },
  { name: 'pulse', options: { duration: 800 } }
]);

Multiple Effects!

Custom Animations

Register and use your own custom animations

Custom Effect! 
BazAnimation.registerAnimation('rainbow', {
  keyFrames: [
    { filter: 'hue-rotate(0deg)' },
    { filter: 'hue-rotate(360deg)' }
  ],
  options: { duration: 2000, iterations: 3 }
}, 'custom');
BazAnimation.animate(element, 'rainbow', {}, ['custom']);
