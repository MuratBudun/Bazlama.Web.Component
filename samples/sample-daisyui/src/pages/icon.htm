Baz-Icon Component

Flexible and accessible icon component designed for easy use of SVG icons

Features

  • 50+ pre-defined Tabler icons
  • Dynamic size, color, and stroke settings
  • Accessibility support (ARIA attributes)
  • Default fallback icon
  • Responsive and customizable

Usage

Basic Usage

<baz-icon icon="search"></baz-icon>

Size Customization

<baz-icon icon="rocket" size="48px"></baz-icon>

Color and Stroke

<baz-icon icon="lightbulb" stroke="red" stroke-width="3"></baz-icon>

Accessibility

<baz-icon 
  icon="search" 
  role="img" 
  aria-label="Search icon"
  aria-hidden="false"
></baz-icon>

Properties

Property Type Default Description
icon string "" Icon name (search, menu2, rocket, etc.)
size string "24px" Icon size (width and height)
stroke string "currentColor" Icon stroke color
stroke-width string "2" Stroke width
role string "img" ARIA role attribute
aria-label string "" Label for accessibility
aria-hidden string "true" Hide from screen readers

TypeScript Example

import BazIcon from "./baz-ui/baz-icon/baz-icon"

// Select component
const icon = document.querySelector('baz-icon') as BazIcon

// Change icon programmatically
icon.icon = "rocket"

// Change size
icon.size = "48px"

// Change color
icon.stroke = "red"

// Change stroke width
icon.strokeWidth = "3"
Icons are from the Tabler Icons library. Provides a consistent and professional look!
