Properties & Attributes

API Reference
Property Type Default Description
icon string "" Icon name from library (e.g., search, home, code)
size string | number "24" Icon size in pixels (width and height)
stroke string "currentColor" Icon stroke color (CSS color value)
stroke-width string | number "2" Stroke width (1-4 recommended)
role string "img" ARIA role attribute for accessibility
aria-label string "" Accessible label for screen readers
aria-hidden string "true" Hide icon from screen readers

TypeScript Integration

Type Safe 
// Import the component
import BazIcon from "./baz-ui/baz-icon/baz-icon"
// Select the icon component
const icon = document.querySelector('baz-icon') as BazIcon
// Change icon dynamically
icon.icon = "rocket"
// Customize appearance
icon.size = "48"
icon.stroke = "#ef4444"
icon.strokeWidth = "3"
// Accessibility
icon.ariaLabel = "Launch rocket"
icon.ariaHidden = "false"
