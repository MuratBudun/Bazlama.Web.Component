Data Table

A feature-rich data table with resizable columns, sorting, and sticky columns

Interactive Playground

Try resizing columns by dragging the column borders. Click on sortable columns to sort. Scroll horizontally to see sticky columns.

Column Resize

Drag column borders to resize. Supports min/max width constraints.

Sortable Columns

Click column headers to sort. Supports ascending and descending order.

Sticky Columns

First N columns stay fixed while scrolling horizontally.

Custom Renderers

Define custom HTML renderers for cell content like badges, icons, etc.

Zebra Striping

Alternating row colors for better readability.

Row Events

Click events on rows with row data in event detail.

Attributes

Attribute Type Default Description
zebra boolean false Enable alternating row colors
compact boolean false Reduce row height for dense data
row-hover boolean false Highlight row on mouse hover
sticky-columns number 0 Number of columns to keep fixed on left

Methods

Method Parameters Returns Description
setColumns() columns: IDataTableColumn[] void Set column definitions
getColumns() - IDataTableColumn[] Get current column definitions
setData() data: T[] void Set table data
getData() - T[] Get current table data
setSort() sort: IDataTableSort | null void Set sort state

Events

Event Detail Description
row-click { row: T, index: number } Fired when a row is clicked
sort-change { column: string, direction: 'asc' | 'desc' } Fired when sort state changes
column-resize { column: string, width: number } Fired when column is resized
