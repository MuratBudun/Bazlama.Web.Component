Bazlama Web Component

A lightweight, decorator-based TypeScript framework for building modern Web Components with reactive properties, automatic DOM updates, and zero dependencies.

Get Started GitHub

Key Features

Decorator-Based API

Use familiar TypeScript decorators like @Property, @Attribute, and @EventAction for clean, readable component code.

Reactive Properties

Automatic DOM updates when properties change. Use @ChangeHooks to define how the UI responds to state changes.

Shadow DOM Support

Choose between Shadow DOM (open/closed) or Light DOM based on your component's needs with simple configuration.

Zero Dependencies

Lightweight and fast. No external runtime dependencies - just pure TypeScript compiled to standard Web Components.

Built-in Hooks

11+ pre-built hooks like useElementText, useElementStyle, useToggleClass for common DOM operations.

TypeScript First

Full TypeScript support with type-safe decorators, intellisense, and compile-time error checking.

Quick Example


import { BazlamaWebComponent, CustomElement, Property, Attribute, ChangeHooks, useElementText } from "bazlama-web-component"

@CustomElement("hello-world")
class HelloWorld extends BazlamaWebComponent {
    constructor() {
        super()
        this.InitBazlamaWebComponent()
    }

    @ChangeHooks([useElementText(".name")])
    @Attribute("name", true)
    @Property({ defaultValue: "World" })
    name: string = "World"

    getRenderTemplate() {
        return `<h1>Hello, <span class="name"></span>!</h1>`
    }
}
