@EventAction

This decorator binds a method to DOM events on elements matching a CSS selector. It automatically handles event listener registration and cleanup. 


@EventAction(query: string, eventName: string)
  • query
    - CSS selector for target element(s).
  • eventName
    - DOM event name (e.g., "click", "input", "change").

Method Signature: The decorated method receives 4 parameters: (eventActionName, element, eventName, event) 


@CustomElement("form-component")
class FormComponent extends BazlamaWebComponent {
    @Property({ defaultValue: "" })
    inputValue: string = ""
    
    // Simple click handler
    @EventAction(".submit-btn", "click")
    onSubmit() {
        console.log("Form submitted:", this.inputValue)
    }
    
    // Full signature with all parameters
    @EventAction(".text-input", "input")
    onInput(
        eventActionName: string,  // "onInput"
        element: HTMLElement,     // The input element
        eventName: string,        // "input"
        event: Event              // The InputEvent
    ) {
        const target = event.target as HTMLInputElement
        this.inputValue = target.value
    }
    
    // Arrow function style (recommended for TypeScript)
    @EventAction(".cancel-btn", "click")
    onCancel = () => {
        this.inputValue = ""
    }
}
  • click - Mouse click
  • dblclick - Double click
  • input - Input value change (fires on every keystroke)
  • change - Value change (fires on blur)
  • keydown / keyup - Keyboard events
  • mouseover / mouseout - Mouse hover
  • focus / blur - Focus events
  • submit - Form submission
