@Attribute

This decorator binds a property to an HTML attribute. When the attribute changes in HTML, the property updates automatically, and vice versa. 


@Attribute(attributeName: string, reflect?: boolean)
  • attributeName
    - The HTML attribute name (use kebab-case).
  • reflect
    - If true, property changes update the attribute. Default: false.

Naming Convention: Use kebab-case for attribute names (e.g., label-text) while the property can use camelCase (e.g., labelText). 


@CustomElement("user-card")
class UserCard extends BazlamaWebComponent {
    // One-way: HTML attribute -> Property
    @Attribute("user-name")
    @Property({ defaultValue: "Anonymous" })
    userName: string = "Anonymous"
    
    // Two-way reflection: Attribute <-> Property
    @Attribute("selected", true)
    @Property({ defaultValue: false })
    selected: boolean = false
}

// HTML Usage:
// <user-card user-name="John" selected="true"></user-card>
        

// Get/set via attribute
const card = document.querySelector('user-card')
card.setAttribute('user-name', 'Jane')
console.log(card.getAttribute('user-name'))  // "Jane"

// Get/set via property (direct access)
card.userName = 'Bob'
console.log(card.userName)  // "Bob"

// With reflection (reflect: true), both stay in sync:
card.selected = true
console.log(card.getAttribute('selected'))  // "true"
