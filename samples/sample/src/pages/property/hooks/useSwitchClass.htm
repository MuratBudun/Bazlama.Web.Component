useSwitchClass

This hook is used to switch CSS classes based on property value changes. It removes the class associated with the old value and adds the class for the new value. 


useSwitchClass(query: string, prefix?: string, suffix?: string)
  • query
    - The query selector to select the element(s).
  • prefix
    - Optional prefix for the class name (e.g., "theme-").
  • suffix
    - Optional suffix for the class name.

Example: With prefix "theme-" and value "dark", the class theme-dark will be added and the previous class (e.g., theme-light) will be removed.

The box above uses useSwitchClass with prefix "theme-". Click the buttons to switch between different theme classes.
