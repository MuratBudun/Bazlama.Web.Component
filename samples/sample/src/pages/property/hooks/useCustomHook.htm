useCustomHook

This hook provides maximum flexibility by allowing you to execute custom logic for each element matching a query. It gives you full access to the element, property value, and component instance. 


useCustomHook(query: string, func: CustomHookFunction)

type CustomHookFunction = (
    target: Element,
    value: TPropertyValueType,
    propertyDefine: PropertyDefine,
    oldValue: TPropertyValueType,
    bazComponent: BazlamaWebComponent
) => void
  • query
    - The query selector to select target elements.
  • func
    - The custom function to execute for each matched element.
  • target
    - The matched DOM element.
  • value
    - The new property value.
  • propertyDefine
    - The property definition object.
  • oldValue
    - The previous property value.
  • bazComponent
    - The component instance.

The rating component above uses useCustomHook to update multiple elements (stars, rating text, and data attribute) with a single custom function.
