useToggleClass

This hook is used to toggle a CSS class based on a boolean property value. When the property is true, the class is added; when false, the class is removed. 


useToggleClass(query: string, className: string)
  • query
    - The query selector to select the element(s).
  • className
    - The CSS class name to toggle.

Note: The class is added when the property value is true (truthy) and removed when it is false (falsy).

The panel above uses useToggleClass to show/hide the content. Click the header to toggle the hidden and rotate-180 classes.

For more information, see the "Script" and "Template" tabs.
