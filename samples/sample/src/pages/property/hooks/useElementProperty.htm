useElementProperty

This hook is used to set the property of an element. It is useful when you want to set the property of an element dynamically. 


function useElementProperty(query: string, property: string, onCalculate?: TPropertyChangeCalculateValue)  

TPropertyChangeCalculateValue = (value: any, query: string, property: string, bazComponent: BazlamaWebComponent) => any
  • query
    - The query selector to select the element.
  • property
    - The property to set.
  • onCalculate
    - A function that calculates the value to set. It is optional.
  • value
    - The current value of the property.
  • query
    - The query selector to select the element.
  • property
    - The property to set.
  • bazComponent
    - The Bazlama Web Component instance.
