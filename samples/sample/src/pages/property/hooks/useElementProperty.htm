useElementProperty

This hook is used to set the property of an element. It is useful when you want to set the property of an element dynamically. 


function useElementProperty(query: string, property: string, onCalculate?: TPropertyChangeCalculateValue)  

TPropertyChangeCalculateValue = (value: any, query: string, property: string, bazComponent: BazlamaWebComponent) => any
  • query
    - The query selector to select the element.
  • property
    - The property to set.
  • onCalculate
    - A function that calculates the value to set. It is optional.
  • value
    - The current value of the property.
  • query
    - The query selector to select the element.
  • property
    - The property to set.
  • bazComponent
    - The Bazlama Web Component instance.

                <baz-s04 volume="0.5"></baz-s04>

In this example, the baz-s04 component wraps a video element. Normally, it is not possible to set the volume of the video element as an attribute.

However, in this example, the volume attribute can be set with the volume attribute. When the volume of the video element is changed, the volume attribute of the baz-s04 component is also changed.

For more information, see the "Script" and "Template" tabs.


                setAttribute('volume', '0.8')
