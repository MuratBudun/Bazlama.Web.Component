useElementAttribute

This hook is used to set the attribute of an element. It is useful when you want to set the attribute of an element dynamically. 


function useElementAttribute(query: string, attribute: string, calcValue?: TCalcValue)  

type TCalcValue = (value: any, query: string, attribute: string, bazComponent: BazlamaWebComponent)
  • query
    - The query selector to select the element.
  • attribute
    - The attribute to set.
  • calcValue
    - A function that calculates the value to set. It is optional.
  • value
    - The current value of the property.
  • query
    - The query selector to select the element.
  • attribute
    - The attribute to set.
  • bazComponent
    - The Bazlama Web Component instance.
