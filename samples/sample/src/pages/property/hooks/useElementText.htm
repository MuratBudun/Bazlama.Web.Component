useElementText

This hook is used to set the text content of an element. It is useful when you want to set the text content of an element from component property on change. 


            useElementText(query: string, prefix: string = "", suffix: string = "")
  • query
    - The query selector to select the element.
  • prefix
    - The prefix to be added to the text content.
  • suffix
    - The suffix to be added to the text content.
