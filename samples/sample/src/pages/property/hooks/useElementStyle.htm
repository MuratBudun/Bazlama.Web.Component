useElementStyle

This hook is used to set inline CSS styles on an element. It is useful when you want to dynamically change the style of an element based on a property value. 


useElementStyle(query: string, style: string, suffix?: string, prefix?: string, onCalculate?: TStyleCalculateValue)

type TStyleCalculateValue = (value: TPropertyValueType, query: string, style: string) => string | number
  • query
    - CSS selector to find target elements.
  • style
    - CSS property name (e.g., "width", "background-color").
  • suffix
    - Optional unit or suffix to append (e.g., "%" or "px").
  • prefix
    - Optional prefix to prepend to the value.
  • onCalculate
    - Optional function to calculate custom value before applying.

Tip: Use onCalculate for dynamic values like color transitions based on thresholds, or complex transformations.
