useElementTextWithFunction

This hook is used to update the text content of an element with a function. The function is called with the current value of the property, the target element, the property define, the old value of the property, and the Bazlama Web Component instance. The function should return the new text content of the element. 


useElementTextWithFunction(
    query: string,
    textUpdater: (
        value: TPropertyValueType,
        target: Element,
        propertyDefine: PropertyDefine,
        oldValue: TPropertyValueType,
        bazComponent: BazlamaWebComponent
    ) => string
)
  • query
    - The query selector to select the element.
  • textUpdater
    - The function to update the text content of the element.
  • value
    - The current value of the property.
  • target
    - The target element.
  • propertyDefine
    - The property define.
  • oldValue
    - The old value of the property.
  • bazComponent
    - The Bazlama Web Component instance.

Turkish: Yukarıdaki baz-s02 bileşeni debit isminde bir property tanımnına sahiptir. Bu property aynı zamanda debit isminde bir html attribute olarak ta tanımlanmıştır. "Change Debit" butonu setAttribute javascript metodu ile bu property'i değiştirir. useElementTextWithFunction hook'u ile debit property'si değiştiğinde debit bilgisi güncellenir. Bu güncelleme işlemi için bir fonksiyon kullanılmıştır. Bu fonksiyon, debit bilgisini alır ve bu bilgiyi formatlar. Daha fazla bilgi için "Script" ve "Template" sekmelerine bakınız.

The baz-s02 component above has a property named debit. This property is also defined as an html attribute named debit. The "Change Debit" button changes this property using the setAttribute javascript method. The useElementTextWithFunction hook updates the debit information when the debit property changes. A function is used for this update process. This function takes the debit information and formats it.

For more information, see the "Script" and "Template" tabs.
