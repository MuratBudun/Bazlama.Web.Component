useElementTextWithFunction

This hook is used to update the text content of an element with a function. The function is called with the current value of the property, the target element, the property define, the old value of the property, and the Bazlama Web Component instance. The function should return the new text content of the element. 


useElementTextWithFunction(
    query: string,
    textUpdater: (
        value: TPropertyValueType,
        target: Element,
        propertyDefine: PropertyDefine,
        oldValue: TPropertyValueType,
        bazComponent: BazlamaWebComponent
    ) => string
)
  • query
    - The query selector to select the element.
  • textUpdater
    - The function to update the text content of the element.
  • value
    - The current value of the property.
  • target
    - The target element.
  • propertyDefine
    - The property define.
  • oldValue
    - The old value of the property.
  • bazComponent
    - The Bazlama Web Component instance.
