useFunction

This hook is the simplest way to add custom logic when a property changes. It executes a function with access to the component instance and property values. 


useFunction(func: TPropertyChangeHook)

type TPropertyChangeHook = (
    bazComponent: BazlamaWebComponent,
    value: TPropertyValueType,
    propertyDefine: PropertyDefine,
    oldValue: TPropertyValueType
) => void
  • func
    - The function to execute when property changes.
  • bazComponent
    - The component instance.
  • value
    - The new property value.
  • propertyDefine
    - The property definition object.
  • oldValue
    - The previous property value.

Tip: Use useFunction for side effects like logging, animations, or triggering external APIs. Combine with other hooks for complex updates.
