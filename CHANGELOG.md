# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- GitHub Actions workflow for automated NPM publishing
- GitHub Actions workflow for automated GitHub Pages deployment
- Comprehensive documentation with live examples
- Interactive samples and demos

## [0.0.5] - 2026-01-18

### Added
- Full TypeScript decorator-based API
- 11+ built-in property hooks
- Reactive property system with automatic DOM updates
- Shadow DOM support (open/closed/none)
- Event action system with automatic cleanup
- Attribute synchronization
- Property change hooks
- Fire event decorator
- Fire render decorator
- Comprehensive test suite
- Multiple sample projects

### Changed
- Improved documentation structure
- Enhanced error handling
- Better TypeScript type definitions

### Fixed
- TypeScript compilation issues
- Hook parameter handling
- Build configuration for multiple environments

## [0.0.1] - Initial Release

### Added
- Basic web component infrastructure
- Property management system
- Initial decorator support
- Core functionality
